## About

**LocalCopy** is an OpenCart OCMOD modifier designed to solve one of the most common problems when installing extensions through the OpenCart Administration panel.

Many hosting providers restrict write permissions to important OpenCart directories. Because of this, uploading an extension through the default installer often results in **permission errors**, **failed installations**, or incomplete uploads.

LocalCopy eliminates these issues by providing a safe installation workflow without requiring insecure permission changes.
